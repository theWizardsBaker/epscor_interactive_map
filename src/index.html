Plot Point 1

The Normalized Difference Vegetation Index (NDVI) is a reflectance index that calculates how healthy vegetation is based on a plant’s absorption and reflectance of light in near-infrared and red regions. This index uses reflectance values at 670 and 800 nanometers (nm) and ranges from -1 to 1. The common range for vegetation is 0.2 to 0.8.
